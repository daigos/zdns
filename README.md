# ZDNS

ZDNS is programmable DNS server and client.  

## Usage

    $ git clone https://github.com/yoshida-eth0/zdns.git
    $ cd zdns
    $ sudo ruby bin/zdnsserver start
    $ sudo ruby bin/zdnsmanager start

You can access http://127.0.0.1:9053 by a browser.
